# csv-zipper

### Docker
You can use docker to run the app locally:
  ` docker-compose up -d`

### Front (React)
- To run :
  ` cd front && yarn run dev`
- To test (react-testing-library + vitest) :
  ` cd front && yarn run test`

### Back (Node.js)
- To run :
  ` cd back && yarn run dev`

### Dependancies
* **antd design**
* react-testing-library
* vitest + vite
* archiver
* multer
* fast-csv
